## 说明

针对单标的CTA类量化策略设计的应用模块，用于实现CTA策略从代码开发、历史回测、参数优化到自动交易的全流程业务功能。

## 安装

安装需要基于2.6.0版本以上的[VN Studio](https://www.vnpy.com)。

直接使用pip命令：

```
pip install vnpy_ctastrategy
```

下载解压后在cmd中运行

```
python setup.py install
```
